消息推送配置

在此配置 Telegram 机器人，用于接收新帖子通知。

如何获取 Bot Token:

  1. 在 Telegram 中搜索并与 @BotFather 对话。
  2. 发送 /newbot 命令，按照提示创建新机器人。
  3. BotFather 会提供一个 Token，格式类似 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11。

如何获取 Chat ID:

  1. 将您的机器人添加到您希望接收消息的群组或频道中。
  2. 在群组/频道中发送一条消息（例如：/my_chat_id）。
  3. 访问浏览器：https://api.telegram.org/bot/getUpdates （将 <YOUR_BOT_TOKEN> 替换为您的机器人 Token）。
  4. 在返回的 JSON 中，找到 "chat" 对象，其中的 "id" 字段就是 Chat ID（通常是一个负数，例如 -1234567890）。
